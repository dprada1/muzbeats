# MuzBeats Documentation

Comprehensive documentation for the MuzBeats project.

## Structure

- **architecture/** - System architecture and design decisions
- **api/** - API documentation and endpoints
- **audio-system/** - Audio player and waveform system documentation
- **search-parser/** - Search parser algorithm and implementation

## Overview

MuzBeats is a modern beat store application with:
- Advanced audio playback system with waveform visualization
- Intelligent search parser supporting musical keys, BPM ranges, and keywords
- Guest checkout with Stripe integration
- Responsive design with lazy loading and performance optimizations
